Apntes Programación Lógica ISES
Ejercicio 5
Código fuente: 
    Algoritmo sin_titulo
        Definir cant_personas, contador Como Entero
        Definir gH, gM Como Entero
        Definir ingreso_genero Como Caracter
        
        Mostrar "Ingese cantidad de personas:"
        leer cant_personas
        
        contador <- 0
        Mientras contador < cant_personas Hacer
            Mostrar "Persona nro. " contador+1 " de " cant_personas
            Mostrar "Ingrese género (H / M):"
            Leer ingreso_genero
            
            Si ingreso_genero == "h" o ingreso_genero == "H" Entonces
                gH <- gH + 1
            FinSi
            
            Si ingreso_genero == "m" o ingreso_genero == "M" Entonces
                gM <- gM + 1
            FinSi
            
            contador <- contador + 1
        FinMientras
        
        Mostrar "Cant. Mujeres: " gM
        Mostrar "Cant. Hombres: " gH
        Mostrar "Porcentaje de mujeres: " (100/cant_personas*gM) " %"
        Mostrar "Porcentaje de hombres: " (100/cant_personas*gH) " %"
    FinAlgoritmo
